# Papers, Peasant! - Unity 6 Project

## Requirements
- Unity 6 (6000.x) — any patch version
- TextMeshPro (auto-imported on first open)

## How to open
1. Open Unity Hub
2. Add project from disk → select this UnityProject/ folder
3. Choose your Unity 6 editor version and click Open
4. Wait for import to finish
5. When "TMP Importer" dialog appears → click "Import TMP Essentials"
6. In the Project panel → Assets/Scenes → double-click MainScene
7. Press Play ▶

## That's it!
The Bootstrap GameObject in the scene has the Game.cs script attached.
Game.cs builds the entire scene at runtime — camera, background, UI, everything.

## How to play (Day 1)
- Click BEGIN DUTY on the title screen
- Read the Head Guard's briefing, click > to advance
- Click RING BELL to call each visitor
- Check their Letter of Passage seal:
  - Emmeloord seal, not expired → Let Through
  - Wrong kingdom seal (Vorn)   → Call Guard  
  - Expired seal                → Deny Entry
- Score is shown as Coins in the top right
